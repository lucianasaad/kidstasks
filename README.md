# kidstasks
Gerenciador de Tarefas Escolares - Projeto React 

O KidsTasks é uma aplicação web desenvolvida para o gerenciamento de tarefas escolares de crianças.
Seu principal objetivo é permitir que pais, responsáveis ou alunos organizem e acompanhem atividades escolares, registrando informações como data de entrega, prioridade e responsável pela tarefa.
O sistema propõe uma interface simples, interativa e moderna construída em React, com funcionalidades de criação, leitura, atualização e exclusão (CRUD).
Além disso, o projeto demonstra o uso de hooks, roteamento e integração com API.


Siga as instruções abaixo para executar o projeto localmente:

1. Instalar dependências:
   
npm install

3. Executar o projeto:

●Rodar a API (JSON Server):

npm run server

●Rodar a aplicação React:

npm run dev

●Ou executar ambos ao mesmo tempo:

npm run dev:all

A aplicação estará disponível em:

Frontend: http://localhost:5173
API: http://localhost:3001/tasks
